# Reader

Next.js frontend for the Immersive Japanese Reader. See [root README](../README.md) for full project documentation.

**Live**: https://learn-a-language.vercel.app

## Quick Start

```bash
cp .env.local.example .env.local  # Add Supabase keys
npm install
npm run dev
```

## Stack

- Next.js 16 (App Router, Server Components)
- React 19
- Tailwind CSS 4
- Supabase JS SDK
- Floating UI (tooltip positioning)
- Vitest + Testing Library
